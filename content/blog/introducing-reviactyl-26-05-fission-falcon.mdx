# A More Modern Foundation

With Reviactyl 26.05, large parts of the platform have been modernized internally. <br />
The backend now runs on Laravel 13, while the frontend has been upgraded to React 19. We have also replaced webpack with Vite, significantly improving development speed and frontend performance. <br />
The admin experience has also evolved with the adoption of FilamentPHP, allowing a cleaner and more maintainable management interface. <br />
These upgrades provide a stronger foundation for future features, extensions, and ecosystem growth. 

<img src="/blog/live-monitoring.png" alt="Reviactyl Dashboard" className="rounded-lg mt-4 h-90 border border-zinc-200 dark:border-zinc-700 invert dark:invert-0" />

# Installation Is Easier Than Ever

Reviactyl 26.05 introduces a new installation process that is more streamlined and user-friendly. <br />
The new Setup Wizard guides administrators through the installation process, helping reduce setup complexity for both beginners and experienced users. <br />
Additionally, we have improved the documentation to provide more comprehensive and easy-to-follow guides for installation, configuration, and troubleshooting. <br />

This allows users to choose the database that best fits their needs and infrastructure.

<img src="/blog/installation-wizard.png" alt="Reviactyl Installer" className="rounded-lg mt-4 h-90 border border-zinc-200 dark:border-zinc-700" />

# Multi-DB Support

Reviactyl 26.05 introduces support for multiple database engines, including **MySQL**, **PostgreSQL**, **SQLite**, and **MariaDB**. <br />
This allows users to choose the database that best fits their needs and infrastructure, providing greater flexibility and compatibility with existing systems. <br />
The installation process has been updated to allow users to select their preferred database engine, and the platform has been optimized to work seamlessly with all supported databases. <br />
This enhancement ensures that Reviactyl can be easily integrated into a wide range of hosting environments and setups, making it more accessible to a broader audience.

# Customization Across the Entire Panel

One of the major focuses of Fission Falcon is personalization. <br />
Users can now configure different navbar and sidebar layouts with four available UI styles. <br />
Server cards can also be displayed in either Grid Mode or Row Mode depending on user preference.

Additional customization improvements include:
- Customizable Sidebar Links
- Customizable Error Pages (404, 403, 500)
- New Avatar Styles including Gravatar and DiceBear options
- RTL Support for right-to-left languages
- New color schemes for better accessibility and contrast
- Larger font sizes and improved spacing for better readability
- Layout options for both the navbar and sidebar, allowing users to choose between different arrangements and styles.

# Categories, Sorting and Server Filters

Reviactyl 26.05 introduces new features for categorizing and sorting servers, making it easier for users to manage and organize their server lists. <br />
Users can now create custom categories and assign servers to them, allowing for better organization and quicker access to specific groups of servers. <br />
Additionally, new sorting options have been added, enabling users to sort their servers by various criteria such as name, creation date, or custom attributes. <br />
These enhancements provide a more efficient and personalized experience for users managing multiple servers within the Reviactyl.

# Expanding the Ecosystem

Reviactyl 26.05 introduces the new Extensions API, opening the door for deeper customization and community-driven integrations.

This release also adds:

- Server Categories
- Egg & Server Category Filters
- Public Server Status Pages
- Multi-Announcements Support
- Historical Server Metrics
- Server and Node Monitoring

Together, these features improve organization, observability, and scalability across larger deployments.

# A Better Experience for Everyone

Fission Falcon introduces many improvements focused directly on end users.

Authentication and account management have been expanded with:

- User Registration System
- Social Login support for Discord, Google, and GitHub
- Turnstile Captcha Support
- Default Language configuration
- GeoLocate language auto-selection

The file manager experience has also improved with:

- File Manager Search
- Video & Image Preview Support
- Force Server Kill functionality for emergency actions

These additions make everyday workflows smoother and more intuitive.

# Introducing Reviactyl Agent

Reviactyl 26.05 officially introduces support for Reviactyl Agent, formerly known as Reviactyl Wings.

This transition represents the beginning of a more unified and scalable infrastructure layer for future Reviactyl deployments.

More improvements around the agent ecosystem will continue throughout the Reviactyl 26 release cycle.

**Pterodactyl Wings users can easily migrate to Reviactyl Agent with our new migration guide in the documentation.** <br />
Note that using Pterodactyl Wings with Reviactyl 26.05 is not recommended and may lead to unexpected issues, so we encourage all users to migrate to Reviactyl Agent as soon as possible. <br />
Migration is a simple process, and our documentation provides step-by-step instructions to ensure a smooth transition. <br />
We are committed to supporting our users throughout this transition and will continue to provide updates and improvements to the Reviactyl Agent ecosystem in the coming months.
