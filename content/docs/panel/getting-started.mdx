<Callout type="warning" title="Please Read Before Proceeding">
  We require **Pterodactyl Panel** to be installed before running Reviactyl Installation.
</Callout>

This Panel is designed to run on your own web server. You will need to have root access to your server in order to run and use this panel.

## Dependencies
Without these the installation will fail.
- `curl`, `tar`, `unzip`, `git`
- **Latest Version of Pterodactyl Panel**
